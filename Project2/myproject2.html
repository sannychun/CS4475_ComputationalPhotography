Original Image

To apply a cartoon filter, we must first determine where all the edges are. Therefore I created a mask that contains all the edges of the given image. The steps to making a mask is as follows:

1. Convert to grayscale
2. Blur image
3. Detect edges using cv2.adaptiveThreshold()
